### Ограничение прав root
В файлы 
- /etc/pam.d/sshd
- /etc/pam.d/login
  
добавлена в начало строка
```
auth required pam_listfile.so item=user sense=deny file=/etc/ssh/denied_users onerr=fail
```

| Параметр          | Пояснение                                                                                          |
| ----------------- | -------------------------------------------------------------------------------------------------- |
| `auth`            | Тип стека PAM: **аутентификация**.                                                                 |
| `required`        | Модуль обязателен; при неудаче доступ запрещён (после всех модулей в стеке).                       |
| `pam_listfile.so` | Модуль: проверка списка в файле.                                                                   |
| `item=user`       | Сравнение по **имени пользователя**, который входит.                                               |
| `sense=deny`      | **Запретить**, если пользователь **найден** в файле.                                               |
| `file=...`        | Путь к файлу со списком.                                                                           |
| `onerr=fail`      | При ошибке чтения файла — **отказ** (строгий вариант; уточните у преподавателя при необходимости). |
